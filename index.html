Welcome to Galactic wars!

Galactic wars is Java simulation of Star Wars battle.

This project is made to show how to use basic principles of Java programming language.

How to use Galactic wars?

To start battle simulation, you have to specify number of soldiers for both Empire and Rebel Alliance.

For optimal results recommended number of soldiers for bith sides is 100, resulting with approximately 30 second simulation. If you want to use, for example, 1000 soldiers, simulation will last about one minute.

All battle simulation logs are written in Croatian language.
